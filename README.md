# Food Store

Aplicación frontend de una tienda de comidas, desarrollada con HTML5, CSS3, TypeScript y Vite.

## Funcionalidades implementadas

- Catálogo de productos con render dinámico
- Búsqueda de productos por nombre en tiempo real
- Filtrado por categoría desde el menú lateral
- Carrito de compras con persistencia en localStorage
- Vista del carrito con nombre, precio, cantidad y total
- Control de cantidad por producto (sumar / restar / eliminar)
- Botón para vaciar el carrito

<br>

## Cómo ejecutarlo

### Requisitos

- Node.js instalado
- pnpm instalado

### Pasos

```bash
# 1. Ir a la carpeta del proyecto
cd frontend

# 2. Instalar dependencias
pnpm install

# 3. Levantar el servidor de desarrollo
pnpm dev
```

La aplicación estará disponible en `http://localhost:5173`.
